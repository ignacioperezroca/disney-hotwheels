# Disney Hotwheels

Legacy Hotwheels interactive archive, preserved as a static site.

## Structure

- `index.php` - main entry
- `juego.php`, `seleccion.php`, `preview-tarjetas.php`, `descarga-poster.php` - preserved experience pages
- `header.php`, `footer.php` - shared PHP fragments
- `img/`, `css/`, `fonts/`, `js/` - original assets and scripts

## Notes
